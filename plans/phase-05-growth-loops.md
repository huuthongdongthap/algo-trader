# Phase 5 — Growth Loops

## Loop 1: Public P&L → Social → Subscribers

**Mechanism:** Post weekly P&L on Twitter/X + Polymarket Discord + CT.
"We trade it ourselves. Here's the proof."

| Metric | Target |
|---|---|
| Activation | Tweet → profile visit → sign-up page |
| Conversion | 2% of profile visitors → Starter trial |
| Retention | P&L transparency = trust = churn < 10%/mo |
| Referral rate | 1 in 5 subscribers shares P&L thread |

**Trigger:** Auto-generate weekly P&L summary from trading logs → queue for manual post.
**Flywheel:** More subs → larger own bankroll → better P&L → more social reach.

---

## Loop 2: Open-Source Engine → GitHub → Community → Paid Signals

**Mechanism:** Release core scanner + analysis engine on GitHub (MIT).
Keep signal delivery (RaaS infra, Telegram bot, Kelly auto-execution) paid.

| Metric | Target |
|---|---|
| Activation | GitHub star → README CTA → hosted tier sign-up |
| Retention | Community PRs improve engine → better signals → subscribers stay |
| Referral rate | 1 in 10 contributors becomes paying user |
| Star → sub conversion | 0.5% (industry baseline) |

**Milestone:** 500 GitHub stars = ~2-3 organic subscribers/week.
**Flywheel:** Open-source credibility → enterprise/quant inbound → Elite tier.

---

## Loop 3: Referral — 20% Revenue Share, 3 Months

**Mechanism:** Every subscriber gets referral link.
Referring sub earns 20% of referred user's subscription for first 3 months.

| Metric | Target |
|---|---|
| Activation | Referral link click → sign-up |
| k-factor | 0.3 initially (each user refers 0.3 users) |
| Retention | Referrer stays active to collect earnings |
| Payback period | 3 months (capped) — CAC = $0 net after referral payout |

**Example:** Referring Starter ($49/mo) earns referrer $9.80/mo × 3 = $29.40 per referral.
**Cap:** 3 months prevents long-term margin erosion.
**Flywheel:** Community → referrals → more signal fees → better infra → better signals.

## Activation Priority

1. Loop 1 — start immediately (only cost: trading capital already deployed)
2. Loop 2 — after 50 paper trades confirm edge (GitHub release = social proof)
3. Loop 3 — after first 10 paying subscribers (need base to refer from)
